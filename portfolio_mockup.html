Pipeline de mejora del modelo aplicado
  • OLS original → R²=0.49%, Breusch-Pagan p=0.043 → heterocedasticidad detectada
  • Transformación log(y) → Breusch-Pagan p=0.066 → varianza estabilizada
  • Box-Cox (λ=1.225) → comparación AIC/BIC con modelo original
  • RLM Huber → coeficientes robustos a outliers
  • Bootstrap (B=1000) → IC percentil para cada coeficiente

Cómo convertir los notebooks en imágenes para el portafolio

1
Agregar plt.savefig()

Añade antes de cada plt.show():
plt.savefig('nombre.png', dpi=200, bbox_inches='tight')

2
Descargar imágenes de Colab

Panel izquierdo → ícono de carpeta → clic derecho sobre cada .png → Descargar.

3
Incrustar en este portafolio

Reemplaza cada viz-placeholder por:
<img src="nombre.png" style="width:100%">

4
Alternativa: exportar a PDF

En Jupyter/Colab: Archivo → Descargar → .ipynb, luego usa nbconvert:
jupyter nbconvert --to pdf notebook.ipynb

5
Publicar en GitHub Pages

Sube portfolio_mockup.html + imágenes a un repo público y activa GitHub Pages para tener URL compartible.

6
Subir notebooks a GitHub

GitHub renderiza los .ipynb automáticamente con todas las salidas visibles, incluyendo las gráficas.

Consejo para reclutadores: 3 opciones de acceso rápido
  • GitHub: Repositorio público con los .ipynb renderizados, un README con capturas y enlace al portafolio.
  • Google Colab: Comparte los notebooks con "Cualquier persona con el enlace puede ver" → el reclutador no necesita instalar nada.
  • Este HTML: Un solo archivo auto-contenido con imágenes embebidas (base64) que funciona sin servidor ni conexión.
